      /* Estilo para el elemento body. */
      body {
        background-color: blue; /* El body tiene fondo color azul. */

Las hojas de estilo controlan la apariencia de la página HTML. Pueden colocarse dentro del HTML o en un archivo aparte. El elemento style de HTML, lleva como contenido hojas de estilo para la página.

Definición de estilos de CSS3

Contiene reglas de estilo mezcladas con comentarios.

Comentario de CSS3

Empieza con /*, termina con */ y puede ocupar 1 o más renglones. Puede colocarse en cualquier lugar de la definición de estilo de CSS3 y es ignorado por el navegador.

Ejemplo:

/* Este es un comentario */

Regla de estilo de CSS3

Las reglas de estilo empiezan con un selector, seguido de una serie de parejas del tipo propiedad: valor separadas por ; y encerradas entre { y }.

Ejemplos:

  • selector {propiedad1: valor1; propiedad2: valor2}
  • body {background-color: blue; color: white}
  • .rosa {color: #fdc8d4}
Selector de estilo de CSS3

Indica a que elementos del HTML se aplican las propiedades de la regla.

Ejemplos:

  • selector
  • body
  • .rosa
Propiedades

Son características predefinidas en el navegador que permiten configurar distintas características visuales de un elemento; para ello se les asignan valores.

Por ejemplo, para definir que el color del texto de un elemento sea azul, se usa:

color: blue

Indican a que elementos del archivo se aplican las propiedades de una regla de estilo de CSS3.

Selector de tipo de elemento

Se usa el tipo de un elemento de HTML. La regla se aplica a todos los elementos de ese tipo que se encuentren en el archivo.

Ejemplo: el selector
h1
se aplica a todos los elementos dentro del archivo, que sean del tipo h1.

Clase

Los elementos de HTML pueden llevar un atributo llamado class, cuyo valor son nombres separados por espacios. A cada uno de esos nombres se les llama clase y pueden usarse en varios elementos dentro del atributo class.

Las clases indican que el elemento tiene una característica especial, la cual se define por medio de hojas de estilos o de JavaScript.

Selector de clase

Es de la forma
.clase
donde clase es el nombre de una clase. La regla se aplica a todos los elementos del archivo que en su atributo class incluyan el nombre de la clase.

Ejemplo: el selector
.rosa
se aplica a todos los elementos dentro del archivo, que incluyan rosa en el valor de su atributo class.

Se usan principalmente las siguientes propiedades:

color

Propiedad que define el color del texto del elemento.

background-color

Propiedad que define el color del fondo del elemento.

Los valores para colores se pueden expresar de alguna de las siguientes formas:

transparent

Color transparente.

Nombre del color

Los nombres de colores que se pueden usar están definidos en https://www.w3.org/TR/css-color-3/#html4. y en https://www.w3.org/TR/css-color-3/#svg-color.

Ejemplo:
blue

rgb(num_rojo, num_verde, num_azul)

Los valores num_rojo, num_verde y num_azul se explican más adelante, pero representan la mezcla de colores rojo, verde y azul en notación decimal.

Ejemplo:
rgb(155, 255, 43)

rgba(num_rojo, num_verde, num_azul, num_transp)

Los valores num_rojo, num_verde, num_azul y num_transp se explican más adelante, pero representan la mezcla de colores rojo, verde y azul en notación decimal, pero aplicándoles un valor para controlar que tan transparente es el color.

Ejemplo:
rgba(155,255,43,0.7)

#rrggbb

Los rr, gg y bb se explican más adelante, pero representan la mezcla de colores rojo, verde y azul en notación hexadecimal.

Ejemplo:
#a2fe09

#rrggbbaa

Los valores rr, gg, bb y aa se explican más adelante, pero representan la mezcla de colores rojo, verde y azul en notación hexadecimal, pero aplicándoles un valor para controlar que tan transparente es el color.

Ejemplo:
#0b467380

num_rojo

Valor decimal entre 0 y 255. Indica la cantidad de rojo que lleva un color.

num_verde

Valor decimal entre 0 y 255. Indica indica la cantidad de verde que lleva un color.

num_azul

Valor decimal entre 0 y 255. Indica la cantidad de azul que lleva un color.

num_transp

Valor decimal fraccionario entre 0 y 1. Indica lo transparente que es un color: 0 - totalmente transparente, 1 - totalmente opaco.

rr

Valor de 2 cifras hexadecimales entre 00 y ff. Indica la cantidad de rojo que lleva un color.

gg

Valor de 2 cifras hexadecimales entre 00 y ff. Indica la cantidad de verde que lleva un color.

bb

Valor de 2 cifras hexadecimales entre 00 y ff. Indica la cantidad de azul que lleva un color.

aa

Valor de 2 cifras hexadecimales entre 00 y ff. Indica lo transparente que es un color. 00 - totalmente transparente, ff - totalmente opaco.

Hay algunas propiedades que expresan tamaños o distancias. Por ejemplo:

font-size

Indica el tamaño de letra.

Los valores son números decimales, que opcionalmente son fraccionales y son seguidos sin espacio por las unidades de medición. Ejemplos: 3em, 2.4rem.

El cero se puede expresar sin unidades; por ejemplo: 0.

Algunas unidades de medición son:

em

La altura de un renglón de texto, usando el tamaño de letra del elemento.

rem

La altura de un renglón de texto, usando el tamaño de letra del elemento raíz (que en HTML, es el elemento html).

vh

1% de la altura del puerto de visión (la ventana).

vw

1% del ancho del puerto de visión (la ventana).

%

Porcentaje del espacio que el elemento padre permite que sus hijos ocupen.

px

Pixeles.

Letra Negrita

Para poner letra en negrita usa la propiedad
font-weight: bold

Tipos de Letra

Usa la propiedad font-family

Cada símbolo que se puede representar en una familia de fuentes se conoce como glifo o pictograma.

Familias Básicas

serif

Es el tipo de letra que lleva adornos.

Cada glifo puede tener un ancho diferente al de los otros de la misma familia.

Permite leer cómodamente textos largos.

Viene predefinida, pero la familia exacta depende de cada navegador web.

sans-serif

Es el tipo de letra que no lleva adornos.

Luce más en títulos, textos cortos e interfaces de usuario.

Cada glifo puede tener un ancho diferente al de los otros de la misma familia.

Viene predefinida, pero la familia exacta depende de cada navegador web.

cursive

Se parece a la letra hecha a mano.

Cada glifo puede tener un ancho diferente al de los otros de la misma familia.

Viene predefinida, pero la familia exacta depende de cada navegador web.

fantasy

Letra decorativa.

Cada glifo puede tener un ancho diferente al de los otros de la misma familia.

Viene predefinida, pero cambia mucho de un navegador a otro.

monospace

Cada glifo tiene el mismo ancho que los otros de la misma familia.

Viene predefinida, pero la familia exacta depende de cada navegador web.

Otras Familias

Cada navegador tiene sus propias familias de fuentes, que pueden ser diferentes a las de otros navegadores.

Cuando el nombre una familia lleva espacios, se debe poner entre apóstrofos ('), o entre comillas (").

Como no sabemos específicamente en cual navegador se abrirá la página, si deseamos usar fuentes que no sean los básicos, hay que poner una lista de familias de un estilo similar, separadas con , y que termine en un font básico de un estilo similar.

Cuando el navegador web ejecuta el código, trata de cargar la primera familia de la lista; si no la tiene, se pasa a la siguiente, y así sucesivamente hasta que puede cargar alguna.

Ejemplo:
"Courier New", Courier, monospace

Descerga Fuentes

Puedes descargar fuentes de algún sitio como Google Fonts. El mismo sitio proporciona instrucciones para obtener una URL que puedes colocar en tu código para que el navegador web descargue la familia que deseas.

Hay otros sitios como google-webfonts-helper que te permite descargar los fuentes para hospedarlos en tu servidor web y que se descarguen junto con tu página. Tienes que crear también un archivo de CSS3 que cargue el font y que debes incluir desde tu archivo HTML. Una vez más, el sitio te ayuda para realizar estas labores.

Con el fuente descargado puedes incluir el nombre de tu familia en el listado de la propiedad font-family.

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Fuentes</title>
    <link
      href="https://fonts.googleapis.com/css2?family=Indie+Flower&display=swap"
      rel="stylesheet">
    <style>
      h1 {
        font-family: sans-serif;
        font-size: 10vh;
      }

      p {
        font-family: Cambria, Cochin, Georgia, Times, 'Times New Roman', serif;
      }

      .bold {
        font-weight: bold;
      }

      .monospace {
        font-family: "Courier New", Courier, monospace
      }

      .cursive {
        font-family: cursive;
      }

      .fantasy {
        font-family: fantasy;
      }

      .descargado {
        font-family: 'Indie Flower', cursive;
      }
    </style>
  </head>
  <body>
    <h1>Fuentes</h1>
    <p><span class=bold>Párrafo</span> con Serif.</p>
    <p class="monospace">Párrafo con monospace.</p>
    <p class="cursive">Párrafo con cursive.</p>
    <p class="fantasy">Párrafo con fantasy.</p>
    <p class="descargado">Párrafo con fuente descargado.</p>
    <footer><p>Copyright &copy; 2020 Gilberto Pacheco Gallegos.</p></footer>
  </body>
</html>
