Acceso Privado

Ejemplo para Navegador que Usa Correctamente el Acceso Privado

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <title>Miembros Privados OK</title>
  </head>
  <body>
    <h1>Miembros Privados OK</h1>
    <script>
      class Cl1 {
        #x;
        saluda() {
          this.#x = 0;
          alert(this.#x);
        }
      }
      const a = new Cl1();
      a.saluda();
    </script>
  </body>
</html>
