Es un control para seleccionar entre un conjunto de opciones.

De forma preddeterminada solo permite seleccionar 0 o una opción.

Cuando el atributo booleano multiple está presente, permite seleccionar 0 o más opciones.

El atributo size indica cuantas opciones mostrar. Por omision solo muestra la opcion seleccionada. Cuando se omite y está el atributo multiple, muestra viarias opciones, pero depende del navegador cuantas mostrar.

Las opciones se indican con el elemento option. El texto de cada opción es el contenido de este elemento. El valor asociado de cada opción (algo así como la llave primaria) se indica con el atributo value.
